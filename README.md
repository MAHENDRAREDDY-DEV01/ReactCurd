# 🚀 React CRUD Application

A modern full-stack CRUD application built with **React**, **Vite**, **Axios**, and **JSON Server**. 
This project demonstrates how to build a complete Create, Read, Update, and Delete workflow using REST APIs and
deploy both the frontend and backend on Render.

## ✨ Highlights

- 📋 View all users
- ➕ Add new users
- ✏️ Edit existing users
- 🗑️ Delete users
- 🔄 Real-time CRUD operations
- 🌐 Fully deployed on Render
- 📱 Responsive and user-friendly interface

## 🛠️ Tech Stack

- ⚛️ React
- ⚡ Vite
- 🌐 Axios
- 🗄️ JSON Server
- 🟢 Node.js
- 🐙 Git & GitHub
- ☁️ Render
